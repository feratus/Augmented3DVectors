Esta actividad explica las operaciones básicas con vectores de manera gráfica usando un sistema de coordenas de referencia en 3D. El usuario puede interactuar con la adición y la resta de dos vectores.

 

Funcionalidad

  • Utilice los controles para modificar los componentes de cada vector
  • Seleccione la operación deseada dentro del menú de opciones (Doble Tap sobre la pantalla para desplegarlo)
  • Mueva el dispositivo sobre el espacio para visualizar los resultados
